# Automobile Sales Dashboard (Plotly + Dash)

## Overview
Interactive dashboard exploring historical automobile sales by **year**, **vehicle type**, and **recession vs non-recession** periods.

## Dataset
Key fields used:
- Year
- Vehicle_Type
- Automobile_Sales
- Recession (indicator)
- (Additional fields used for filtering/aggregation)

## Features
- Year selection
- Recession vs non-recession view
- Vehicle-type comparisons
- (Any other controls you built)

## Tech Stack
Python, Pandas, Plotly, Dash

1. Clone this repo
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
3. Run:
   python app.py
4. Open the local URL shown in your terminal


## Key Insights
- Sales tend to decline during recession periods across most vehicle categories
- Sales trends vary by vehicle type and over time
- Segmentation (year + vehicle type) reveals patterns hidden in overall totals
